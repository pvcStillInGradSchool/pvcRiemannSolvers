Detailed Description

template<typename P, typename R>
class mini::spatial::fr::Lobatto< P, R >

A specialized version of FR using a Lagrange expansion on Lobatto roots with the "Lumping Lobatto" correction function.

The \( g_\mathrm{right} \) only corrects the flux divergence at the rightest solution point, which is a flux point.
