Detailed Description

template<std::floating_point T, int N>
struct mini::integrator::Lobatto< T, N >

Gauss–Lobatto quadrature rules, i.e. \( \int_{-1}^{1} f(\xi) \,\mathrm{d}\xi \approx \sum_{q=1}^{Q} w_q f(\xi_q) \), in which \( \xi_1 = -1 \) and \( \xi_Q = +1 \).

Template Parameters
TType of scalar variables.
NNnumber of quadrature points, which must be chosen from { 2, 3, 4, 5, 6 }.
