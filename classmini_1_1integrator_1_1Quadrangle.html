template<int kPhysDim, class Gx, class Gy>
class mini::integrator::Quadrangle< kPhysDim, Gx, Gy >

Numerical integrators on quadrilateral elements.

Template Parameters
kPhysDimDimension of the physical space.
GxThe quadrature rule in the \(\xi\) direction.
GyThe quadrature rule in the \(\eta\) direction.
