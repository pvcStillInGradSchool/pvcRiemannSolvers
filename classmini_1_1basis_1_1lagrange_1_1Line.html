Detailed Description

template<std::floating_point S, int kDegree>
class mini::basis::lagrange::Line< S, kDegree >

The Lagrange basis functions on the standard Line element.

The Lagrange polynomal at the \(k\)-th node \( L_k(\xi) \) is defined as \( \prod_{l=0,l\ne k}^{N-1}\frac{\xi-\xi_{l}}{\xi_{k}-\xi_{l}} \).

Template Parameters
Sthe type of nodes
kDegreethe degree of each Lagrange polynomial
