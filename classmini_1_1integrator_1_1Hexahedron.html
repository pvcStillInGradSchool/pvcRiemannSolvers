Detailed Description

template<class Gx, class Gy, class Gz>
class mini::integrator::Hexahedron< Gx, Gy, Gz >

Numerical integrators on hexahedral elements.

Template Parameters
GxThe quadrature rule in the \(\xi\) direction.
GyThe quadrature rule in the \(\eta\) direction.
GzThe quadrature rule in the \(\zeta\) direction.
