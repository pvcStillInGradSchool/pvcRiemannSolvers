Detailed Description

template<std::floating_point S, int kDegreeX, int kDegreeY, int kDegreeZ>
class mini::basis::lagrange::Hexahedron< S, kDegreeX, kDegreeY, kDegreeZ >

The Lagrange basis functions on the standard Hexahedron element.

The Lagrange polynomal at the \((i,j,k)\)-th node \( L_{i,j,k}(\xi, \eta, \zeta) \) is defined as \( L_i(\xi)\,L_j(\eta)\,L_k(\zeta) \).

Template Parameters
Sthe type of coordinates
kDegreeXthe degree of each \( L_i(\xi) \)
kDegreeYthe degree of each \( L_j(\eta) \)
kDegreeZthe degree of each \( L_k(\zeta) \)
