Detailed Description

template<class Gx, class Gy, class Gz, int kComponents, bool kL = false>
class mini::polynomial::Hexahedron< Gx, Gy, Gz, kComponents, kL >

A vector-valued function interpolated on an given basis::lagrange::Hexahedron basis.

The interpolation nodes are collocated with quadrature points.

Template Parameters
GxThe quadrature rule in the 1st dimension.
GyThe quadrature rule in the 2nd dimension.
GzThe quadrature rule in the 3rd dimension.
kComponentsThe number of function components.
kLFormulate in local (parametric) space or not.
