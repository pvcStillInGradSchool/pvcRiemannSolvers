Detailed Description

template<std::integral Int>
class mini::mesh::metis::Mesh< Int >

A CSR (Compressed Sparse Row) representation of meshes.

In this representation, the global indices of nodes in the ith cell are [ nodes(range(i)), nodes(range(i) + 1), ..., nodes(range(i + 1)) ).
