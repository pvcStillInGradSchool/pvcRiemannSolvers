Detailed Description

template<class Gx, class Gy, int Qz>
class mini::integrator::Pyramid< Gx, Gy, Qz >

Numerical integrators on pyramidal elements.

Template Parameters
GxThe quadrature rule in the \(\xi\) direction.
GyThe quadrature rule in the \(\eta\) direction.
QzNumber of qudrature points in the \(\zeta\) direction.
