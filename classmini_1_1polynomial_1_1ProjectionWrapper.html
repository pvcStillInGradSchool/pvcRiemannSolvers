Detailed Description

template<std::floating_point S, int kDimensions, int kDegrees, int kComponents>
class mini::polynomial::ProjectionWrapper< S, kDimensions, kDegrees, kComponents >

A light-weighted wrapper of Projection.

The object is light-weighted in the sense that it holds a pointer to a basis::OrthoNormal object and assumes the basis does not change in the lifetime of this object.

Template Parameters
Sthe data type of scalar components
kDimensionsthe dimension of the underlying physical space
kDegreesthe degree of completeness
kComponentsthe number of function components
