Detailed Description

template<std::floating_point Scalar, int kPhysDim, int kPoints>
class mini::integrator::Triangle< Scalar, kPhysDim, kPoints >

Numerical integrators on triangular elements.

Template Parameters
ScalarType of scalar variables.
kPhysDimDimension of the physical space.
kPointsNumber of qudrature points, which must be chosen from { 1, 3, 6, 12, 16 }.
